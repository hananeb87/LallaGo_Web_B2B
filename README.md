# LallaGo — Corporate Mobility as a Service

**Morocco's first CSRD-compliant corporate mobility platform.**

One app for executive transport, commuter shuttles, and ESG reporting.

---

## Features

- 🚗 Licensed transport partners
- 📊 Automatic ESG reporting (CSRD-aligned)
- ⚡ Mobility budget management
- 🌍 Multi-city: Casablanca · Marrakech · Rabat (coming soon)
- 🔋 EV-first fleet roadmap (Scope 3 reduction)
